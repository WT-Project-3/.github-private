# 🍽️ FreshDine – Smart Meal Planning Web Application

FreshDine is a user-centered web application designed to help individuals discover recipes, plan meals, and manage their dietary preferences in a simple and intuitive way.

This project was developed as part of a Web Technology course, focusing on usability, user experience design, and practical web development.

---

## 🚀 Features

- 🔐 User Authentication (Register / Login)
- 👤 User Profile Management
- 🥗 Dietary Preference Selection
- 🔍 Recipe Discovery
- 📖 Detailed Recipe View (ingredients, instructions)
- 📅 Meal Planning Functionality
- 🛒 Checkout & Delivery Concept (prototype level)

---

## 🎯 Project Goals

The main goal of FreshDine is to:
- Simplify meal planning for users with different dietary needs
- Provide an intuitive and user-friendly interface
- Demonstrate full web development workflow from concept to prototype
- Apply software engineering concepts such as user personas, scenarios, and use cases

---

## 🧑‍💻 Target Users

- Students and busy individuals
- People with dietary restrictions (vegetarian, halal, etc.)
- Users looking for simple meal planning solutions

## 🧩 System Overview

FreshDine allows users to:
1. Create an account and set preferences
2. Discover recipes tailored to their needs
3. View detailed cooking instructions
4. Plan meals for the week
5. Proceed to a simulated checkout and delivery process
